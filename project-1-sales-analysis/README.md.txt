# Sales Performance Analysis

## Project Overview
This project analyzes sales data from a retail Superstore to gain insights on sales performance across regions, categories, and products. The goal is to practice core data analyst skills using Python and Pandas.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
- Source: Superstore Sales Dataset (CSV)
- 9800 rows, 18 columns

## Key Analysis
- Total Sales: 2252607.4127
- Region with highest sales: West
- Most selling category: Technology
- Top product by sales: Canon imageCLASS 2200 Advanced Copier
- Monthly sales trend: Peak sales in November, lowest in February


## How to Run
1. Open `sales_analysis.ipynb` in Jupyter Notebook
2. Run all cells sequentially
3. Explore charts and analysis
